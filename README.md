# DuckShoot #
This game was developed in x8086 16-bit Assembly Language.
### HOW TO PLAY? ###
Create an ASM file and paste the code
<br>Use VSCode with MASM/TASM extension</br>
Run ASM code
<br>Enjoy :)</br>
